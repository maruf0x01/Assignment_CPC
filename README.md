# Programming Club Assignment

## Overview
This repository contains various C++ projects developed as part of the programming club assignment. Each project demonstrates different algorithms and data structures, designed to solve specific problems and showcase the implementation of various concepts in C++.

## Projects

### 1. Ambigious
- **File:** `Ambigious.cpp`
- **Description:** A project that explores ambiguous situations in problem-solving.

### 2. Budget Manager
- **File:** `Budget_Manager.cpp`
- **Description:** A tool for managing budgets and expenses effectively.

### 3. Greedy Input
- **File:** `Greedy_Input.cpp`
- **Description:** Demonstrates greedy algorithms and their applications.

### 4. Immortal Player
- **File:** `Immortal_Player.cpp`
- **Description:** A project that simulates an immortal player in a game environment.

### 5. Locked Room
- **File:** `Locked_Room.cpp`
- **Description:** A puzzle-solving project that involves escaping a locked room.

### 6. Namespace
- **File:** `Namespace.cpp`
- **Description:** An exploration of namespaces in C++ and their usage.
